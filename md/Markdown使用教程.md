# Markdown使用教程

## 什么是markdown？

Markdown 是一种轻量级标记语言，用于将普通文本格式化为结构化文档。

其设计目的是使文档的书写和阅读更加简便。

Markdown 主要用于撰写说明文档，简洁的文本格式使得它在开发者和写作者中广受欢迎。

### 适用人群

1. **开发者**：Markdown 常用于编写代码注释、项目文档（如README文件）、API文档等。
2. **写作者和博主**：Markdown 简洁的语法非常适合写作和博客文章发布。
3. **学生和研究人员**：用于笔记、论文大纲、研究项目文档等。
4. **项目经理和团队协作人员**：用于项目计划、会议记录、任务跟踪等。

### 适用场景

1. **文档编写**：项目文档、技术手册、使用说明、教程等。
2. **笔记记录**：个人笔记、课堂笔记、会议记录等。
3. **博客和网站内容创作**：许多博客平台（如GitHub Pages, Jekyll）都支持Markdown。
4. **版本控制系统**：如GitHub, GitLab的README文件和其他文档。
5. **电子邮件**：许多邮件客户端支持Markdown，便于编写结构化邮件。
6. **论坛和在线社区**：许多论坛和社区（如Reddit）支持Markdown，用于帖子和评论格式化。

### 优点

1. **易学易用**：语法简单明了，学习曲线平缓，几分钟就能掌握基本用法。
2. **纯文本格式**：使用任何文本编辑器都可以编写和编辑，不依赖特定的软件。
3. **可读性强**：即使在未渲染状态下，Markdown 文档也易于阅读和理解。
4. **可移植性**：支持多种平台和工具，可以轻松转换为HTML、PDF等多种格式。
5. **高效写作**：无需复杂的格式设置，专注于内容创作，提高写作效率。
6. **无干扰写作**：避免了复杂的排版工具干扰，专注于内容本身。

## markdown语法介绍

### 标题

在文本前输入：**# + 空格**。一个井号代表一级标题，两个井号二级标题，以此类推，一共有六级标题

```
# this is h1
## this is h2
### this is h3
#### this is h4
##### this is h5
###### this is h6
```

效果：

# this is h1

## this is h2

### this is h3

#### this is h4

##### this is h5

###### this is h6

### 字体

```
**加粗字体**

__加粗字体__

*倾斜字体*

_倾斜字体_

***加粗以及倾斜***

~~删除线~~
```

**加粗字体** 

**加粗字体**

*倾斜字体*

*倾斜字体* 

***加粗以及倾斜***

~~删除线~~

### 分割线

要创建分隔线，请在单独一行上使用三个或多个星号 (`***`)、破折号 (`---`) 或下划线 (`___`) ，并且不能包含其他内容。可以在星号或是减号中间插入空格。下面每种写法都可以建立分隔线：

```text
* * *
***
**********
- - -
_________________
```

### 引用

要创建块引用，请在段落前添加一个 `>` 符号。 

```text
> Dorothy followed her through many of the beautiful rooms in her castle.
>
> The Witch bade her clean the pots and kettles and sweep the floor and keep the fire fed with wood.
```

效果如下：

> Dorothy followed her through many of the beautiful rooms in her castle.
> 
> The Witch bade her clean the pots and kettles and sweep the floor and keep the fire fed with wood.

区块引用可以嵌套（例如：引用内的引用），只要根据层次加上不同数量的 > ：  

```text
> Dorothy followed her through many of the beautiful rooms in her castle.
>
>> The Witch bade her clean the pots and kettles and sweep the floor and keep the fire fed with wood.
```

效果如下：

> Dorothy followed her through many of the beautiful rooms in her castle.
> 
> > The Witch bade her clean the pots and kettles and sweep the floor and keep the fire fed with wood.

### 列表

Markdown 支持有序列表和无序列表。 无序列表使用星号、加号或是减号作为列表标记。 示例：

```text
- 列表内容
+ 列表内容
* 列表内容

注意：- + * 跟内容之间都要有一个空格
```

效果如下：

- 列表内容
+ 列表内容
* 列表内容

有序列表则使用数字接着一个英文句点作为标记。 示例：

```text
1. 列表内容
2. 列表内容
3. 列表内容

注意：序号跟内容之间要有空格
```

效果如下： 

1. 列表内容 

2. 列表内容 

3. 列表内容

*你在列表标记上使用的数字并不会影响输出的 HTML 结果。*

```text
你的标记写成：
1.  Bird
1.  McHale
1.  Parish

甚至：

8. Bird
1. McHale
4. Parish

效果都一样。
```

列表可以嵌套，上一级和下一级之间敲三个空格即可。

```text
* 一级无序列表内容
   * 二级无序列表内容
   * 二级无序列表内容
   * 二级无序列表内容
```

效果如下： 

- 一级无序列表内容
  - 二级无序列表内容
  - 二级无序列表内容
  - 二级无序列表内容

### 表格

| 内容 | 内容 | 内容 |
```

|:--- |:--- |:--- |
| 内容  | 内容  | 内容  |
| 内容  | 内容  | 内容  |
| 内容  | 内容  | 内容  |

### 代码

在Markdown中加入代码块有三种方式：

1.缩进 4 个空格

```text
    <html>
      <head>
      </head>
    </html>
```

效果如下：

```
<html>
 <head>
 </head>
</html>
```

2.将代码包裹在反引号 (`` ` ``) 中

```text
这里有一句代码`代码内容`。
```

效果如下： 这里有一句代码`代码内容`。  

3.代码之间分别用三个反引号（(` ``` `）或三个波浪号（~~~）包起来

```text

```

{
  "firstName": "John",
  "lastName": "Smith",
  "age": 25
}

```

```

效果如下：

```text
{
 "firstName": "John",
 "lastName": "Smith",
 "age": 25
 }
```

还可以在上面的 ``` 后面注明你的代码类型，可以产生相应的代码高亮。

### 链接语法

链接文本放在中括号内，链接地址放在后面的括号中，链接title可选

```
这是一个链接 [Markdown语法](https://markdown.com.cn)。
```

这是一个链接 [Markdown语法](https://markdown.com.cn)。

#### 网址和Email地址

使用尖括号可以很方便地把URL或者email地址变成可点击的链接。

```
<https://markdown.com.cn>
<fake@example.com>
```

渲染效果如下：

[https://markdown.com.cn](https://markdown.com.cn/)  
[fake@example.com](mailto:fake@example.com)

#### 引用类型链接

引用样式链接是一种特殊的链接，它使URL在Markdown中更易于显示和阅读。参考样式链接分为两部分：与文本保持内联的部分以及存储在文件中其他位置的部分，以使文本易于阅读。

```text
This is [an example][1] reference-style link.
```

```text
[1]: http://example.com/  "Optional Title Here"
```

渲染效果：

This is [an example][1] reference-style link.

[1]: http://example.com/ "Optional Title Here"

```

```

### 脚注

在方括号（`[1]`）内添加插入符号和标识符。标识符可以是数字或单词，但不能包含空格或制表符。

在括号内使用另一个插入符号和数字添加脚注，并用冒号和文本（`[1]: My footnote.`）。脚注可以在除列表，块引号和表之类的其他元素之外的任何位置。

```
Here's a simple footnote,[2] and here's a longer one.[bignote]

[2]: This is the first footnote.

[bignote]: Here's one with multiple paragraphs and code.

    Indent paragraphs to include them in the footnote.

    `{ my code }`

    Add as many paragraphs as you like.
```

渲染效果：

Here's a simple footnote,[2] and here's a longer one.[bignote]

 [2]: This is the first footnote.

 [bignote]: Here's one with multiple paragraphs and code. Indent paragraphs to include them in the footnote. `{ my code }` Add as many paragraphs as you like.

### 任务列表语法

要创建任务列表，请在任务列表项之前添加破折号`-`和方括号`[ ]`，并在`[ ]`前面加上空格。

```
- [x] Write the press release
- [ ] Update the website
- [ ] Contact the media
```

渲染效果：

- [x] Write the press release
- [ ] Update the website
- [ ] Contact the media
